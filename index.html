4장 변수와 스코프, 메모리

  • 자바스크립트 변수는 느슨한 타입을 취함 - 특정 시간에서 변수는 특정 값을 가리키는 이름일 뿐임

4.1 원시 값과 참조 값

  • ECMAScript의 변수는 원시 값과 참조 값 두가지 타입의 데이터를 저장가능
  • 원시 값 - 단순한 데이터로 Undefined,Null,불리언,숫자,문자열
  • 참조 값 - 여러 값으로 구성되는 객체
원시 값과 참조 값의 차이
  • 참조 값을 다룰 때는 언제든 프로퍼티와 메서드를 추가하거나 바꾸고 삭제 가능
  • 원시 값에는 프로퍼티가 없지만 추가하려 해도 에러가 생기지 않음 - 하지만 접근하려 하면 undefined 반환
  • 참조 값을 변수에서 다른 변수로 복사하면 힙에 저장된 객체를 가리키는 포인터가 복사됨

1장 자바스크립트란 무엇인가?

자바스크립트의 간단한 역사
  • 1995년 자바스크립트 처음 등장했을 때 주요 목적은 입력 유효성 검사
  • 넷스케이프에서 일하던 브랜든 아이흐(Breadan Eich)가 Mocah, 나중엔 LiveScript라 불리는 스크립트 언어 개발
  • 경쟁사인 마이크로소프트에서 JScript 라는 이름의 자바스크립트를 포함한 IE3 출시
  • 자바스크립트 표준화 필요성 증가
  • 1997년 자바스크립트 1.1이 ECMA(유럽컴퓨터제작자협회)에 제안
  • JavaScript 1.1 을 손질해 ECMA-262 를 만들었고 이 표준은 ECMAScript(에크마스크립트)라는 이름의 스크립트 언어를 정의
자바스크립트 구현은 다음 세가지로 나뉨
  • 코어 (ECMAScript) : 단순히 명세에서 설명하는 부분들을 구현한 언어 (문법, 타입, 선언문, 키워드, 예약어, 연산자, 객체)
  • 문서 객체 모델 (DOM) : XML을 HTML에서 사용할 수 있도록 확장한 API
  • 브라우저 객체 모델 (BOM)
